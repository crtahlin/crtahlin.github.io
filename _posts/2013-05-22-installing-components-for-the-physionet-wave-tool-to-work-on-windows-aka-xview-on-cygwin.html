After trying to figure out how to use WAVE on my Windows box running Cygwin, I will try to recap the necessary conditions here. (I have not retested the procedure, it might be that some of the previous steps I did according to "official" instructions are also necessary).

The "official" instructions can be found here:
http://www.physionet.org/physiotools/xview/

But they seem to be somewhat obsolete as far as Windows and Cygwin is concerned.

Do the following:

  1. Install the Cygwin packages that are requested by the instructions:
    sunrpc, libX11-devel, X-start-menu-icons and ...
  2. ... while your are in Cygwin setup, also select the following packages for installation: 
    xview, xview-devel     (Ignore xview package for Windows in the official instructions.)
  3. I am not sure if they are needed, but add (as in official instructions) the following lines to .bashrc and .bash_profile (which can be found in your Windows home directory or you need to create them):
    export PATH=/usr/openwin/bin:$PATH
    export DISPLAY=:0.0
  4. Run the X server with:
    startxwin(The startxwin.bat file mentioned in the instructions is nowhere to be found; but this seems to work nevertheless.)
Now you can test if your WAVE app works by following the instructions here:
http://www.physionet.org/physiotools/wug/node14.htm

Fingers crossed!
