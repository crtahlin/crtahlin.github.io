Some Slovenian banks had their bad credits transferred to the Bank Asset Management Company (BAMC)  a.k.a. "bad bank". The information about the credits was made public.

Specifically, this post contains links (on 24.7.2014) to web pages containing data about each contract transferred to the bad bank from the NLB and NKBM banks, who was in the management and advisory boards at the time, etc. (The English version of the page does not seem to work at the moment, hence the link points to the Slovenian version.)

As an exercise in webscraping, motivated by the Coursera "Getting and Cleaning Data" course, I wrote some code that scrapes the data published in HTML tables, parses it and imports it into an R data frame.

The code is in this gist. (I would include it in the blog, but it seems the blogger+gist integration does not work in certain cases.)

The function getCreditData() takes as an argument the name of the bank ("NLB" or "NKBM") and returns a data frame with the columns published on the BAMC website.

I might do some visualizations of the data in the future (a couple of ggplots are at the bottom of the code file, just to get a feeling).
