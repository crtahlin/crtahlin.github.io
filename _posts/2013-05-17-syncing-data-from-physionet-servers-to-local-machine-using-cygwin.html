Full instructions for downloading physionet data can be found here:
http://physionet.org/faq.shtml#downloading-databases

Below is the procedure that worked for me. List the available databases and their sizes with the command: rsync physionet.org:: 

Use cygwin. Create a directory to hold the copy of the database, e.g.:
mkdir /usr/database 
mkdir /usr/database/sddb 

Use the rsync command to copy the database to the newly created directory:
rsync -Cavzr physionet.org::sddb /cygdrive/c/cygwin/usr/database/sddb
