I have been doing some development with the shiny package for R. The package runs a web server on the localhost and enables the user to connect to it via a web browser. The web page is dynamic ("reactive") in the sense that the user can choose certain options and the page updates its output. The output is, of course,  generated by R.

As a personal mental exercise, I will list the structure of the files/functions used by the server (as I am already going through the package documentation). Correctness and completeness not guaranteed, comments welcome.
